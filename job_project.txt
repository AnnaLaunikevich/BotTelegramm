Бот помогает найти школы нужного района и предоставляет ссылку на сайт школы, которую выбрал пользователь

Для работы необходимо установить библиотеку aiogram
После этого в проекте бота создаем два файла bot.py и config.py

В файл config.py мы записываем токен, полученный от @BotFather

Файл bot.py является основным файлом. В него мы импортируем необходимые модули библиотеки aiogram
и бота, а так же инициализируем объекты бота и диспетчера:
from aiogram import Bot, types - импортирует класс бота и типы данных в функциях;
from aiogram.dispatcher import Dispatcher - импортирует класс диспетчера, который реагирует на сообщения
пользователя;
from aiogram.utils import executor - для запуска и выхода бота онлайн.

bot = Bot(token=TOKEN) - инициализирует бота
dp = Dispatcher(bot) - инициализирует диспетчера

Записываем декоратор и далее функцию.
Т.К. aiogram асинхронная библиотека, перед def пишем async.
Создаем первую функцию начала работы бота.
Вторая функция создается для реагирования бота на сообщения пользователя.
Ее мы заполняем списком школ нужного района, а затем при выборе школы - создаем кнопку,
которая перенаправляет пользователя
на сайт выбранной школы.
В конце пишется программа запуска бота - executor.start_polling(dp)


Для упрощения работы пользователя с ботом создаем кнопки-клавиатуры и инлайн-кнопки.
Для этого в проекте БотТелеграм создаем директорий keyboards.
Создаем 2 файла: init и keyboard.
Init - необходим для импортирования кнопок в основной файл bot.py. Импортируем все необходимые кнопки.
Keyboard - для создания кнопок.
В файл Keyboard импортируем классы кнопок.
Для создания кнопки-клавиатуры отдельно служит класс KeyboardButton.
Для замены одной кнопки-клавиатуры несолькими - служит класс ReplyKeyboardMarkup.
Для инлайн-кнопки соответственно InlineKeyboardButton и InlineKeyboardMarkup.
Создаем необходимые кнопки.

Бот готов.

В терминате пишем  python bot.py и бот запускается.
